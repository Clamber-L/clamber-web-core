🚀 Kafka API 代理服务器

📡 Kafka API 端点

以下端点将转发到 Kafka Example API (端口 3000):

GET /api/kafka/
Kafka API 根路径
GET /api/kafka/health
健康检查
POST /api/kafka/send-message
发送消息
POST /api/kafka/send-user-message
发送用户消息
GET /api/kafka/producer-stats
获取生产者统计
GET /api/kafka/consumer-stats
获取消费者统计

⚙️ 配置 API 端点

以下端点将转发到 Kafka Config Example API (端口 3001):

GET /api/config/
配置 API 根路径
GET /api/config/health
健康检查
POST /api/config/send-message
发送消息
